node.js 기본

1.Ajax 개요

XMLHttp Request 객체를 이용해서 전체 페이지를 새로 고치지 않고도 페이지의 일부만을 위한 데이터를 로드하는 기법이며 한 마디로 자바스크립트를 사용한 비동기 통신 클라이언트와 서버간에 XML 데이터를 주고받는 기술

2.데이터 전송방식

서버와 클라이언트가 데이터를 주고 받을 때 맞춰야하는 형식

2-1 CSV 형식

많은양의 데이터를 전송할 때 활용하면 좋은 방식

금교초등학교 1학년 10번 홍길동 도농초등학교 2학년 9번 김철수

2-2 XML 형식

HTML 형식처럼 태그로 데이터를 표현하는 방식

			
				홍길동
				금교초등학교
				10번

2-3 JSON 형식

CSV 형식과 XML 형식의 단점을 모두 극복한 형식 (무조건 큰따운표 사용)

[{ "name": 홍길동, "school" : 금교초등학교, "number" : 10번 }]

2-4 전송형식 장단점

CSV : 용량이 적다 / 가독성이 떨어진다. XML : 가독성이 좋다. / 용량이 크다. , 데이터의 양이커지면 분석속도가 떨어진다. JSON : 용량이 적다. , 가독성이 좋다 / 데이터 양이 커지면 분석속도가 떨어진다.
