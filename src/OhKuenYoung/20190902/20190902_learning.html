문서 객체 조작

기존의 자바스크립트만으로 문서 객체를 다루려면 정말 복잡하지만 jQuery를 사용하면 간단하게 적용할 수 있다.

2.문서 객체의 클래스 속성 추가

addClass() : 객체의 클래스 속성을 추가할 때 사용하는 메서드

	$(document).ready(function(){
		$('p').addClass('item');
	});

3.문서 객체의 클래스 제거

removeClass() : 객체의 클래스 속성을 제거할 때 사용하는 메서드

	$(document).ready(function(){
		$('p').removeClass('item');
	});

4.문서 객체의 속성

attr() : 속성과 관련된 모든 기능을 수행합니다.

속성 검사 
	$(document).ready(function ex03(){
		var tit = $('.test').attr('title');

		alert(tit);
	});
속성 추가
  • 1. $(selector).attr(name, value);
  • 2. $(selector).attr(name, function(index, attr){});
  • 3. $(selector).attr(object);
    $(document).ready(function ex04(){
			$('.attr_ex01').attr('title', HI);
		});
	
    $(document).ready(function ex05(){
			$('.attr_ex01').attr('title', function (index) {
        return (index + 1) + 'hi';
      });
		});
	
    $(document).ready(function ex06(){
			$('.attr_ex01').attr({
        title: function (index) {
            return (index + 1) + 'hi';
        },
        value: 'hi2';
      });
		});

5.속성 제거

removeAttr(name) 문서 객체의 속성을 제거

    $(document).ready(function ex07(){
      $('.attr_ex01').removeAttr('title');
    });

10.문서 객체 삽입, 이동

  • $(A).appendTo(B) : A를 B의 뒷부분에 추가
  • $(A).prependTo(B) : A를 B의 앞부분에 추가
  • $(A).insertAfter(B) : A를 B의 뒷부분에 추가
  • $(A).insertBefore(B) : A를 B의 앞부분에 추가
  • $(A).append(B) : B를 A의 뒷부분에 추가
  • $(A).prepend(B) : B를 A의 앞부분에 추가
  • $(A).After(B) : B를 A의 뒷부분에 추가
