window 객체

screen 객체

운영체제 화면의 속성을 갖는 객체 (웹브라우저의 화면 X)

  var child = window.open('','','width=300, height=200');
  var width = screen.width;
  var height = screen.height;

  child.moveTo(0, 0);
  child.resizeTo(width, height);

  setIntervar (function () {
    child.resizeBy(-20, -20);
    child.moveBy(10, 10);
  }, 1000 );

객체의 노드 생성 메서드

  • createElement(tagName) : 요소노드 생성
  • createTextNode(text) : 텍스트노드 생성

문서 객체의 연결 메서드

appendChild(node)

    var header = document.createElement('h1');
    var textNode = document.createTextNode('hello DOM');

    header.appendChild(textNode);
    document.body.appendChild(header);

문서 객체의 속성 메서드

  • setAttribute(name, value) : 객체의 속성을 지정
  • getAttribute(name) : 객체의 속성을 가져옴
    var atag = document.createElement('a');
    var atagtext = document.createTextNode('hello DOM');

    atag.appendChild(atagtext);
    atag.setAttribute('href', 'www.naver.com');

    document.body.appendChild(atag);

문서 객체의 선택 메서드

  • getElementById(id) : 태그의 ID 속성이 ID 매개변수와 일치하는 문서 객체를 가져옴
  • getElementByName(name) : 태그의 name 속성이 name 매개변수와 일치하는 문서 객체를 배열로 가져옴
  • getElementTagName(tagName) : 태그의 name 속성이 name 매개변수와 일치하는 문서 객체를 배열로 가져옴
  • querySelector(선택자) : 선택자로 가장 처음 선택되는 문서 객체를 가져옴
  • querySelectorAll(선택자) : 선택자를 통해 선택되는 문서 객체를 배열로 가져옴
안녕 안녕 
    span id="test" name="test2" class="test3"안녕 span
    span class="test3" 안녕 span

    var getID = document.getElementById('test');
    var getName = document.getElementsByName('test2');
    var getTag = document.getElementsByTagName('span');
    var query = document.querySelector('.test3');
    var queryall = document.querySelectorAll('.test3');

    window.alert('getElementById : ' + getID);
    window.alert('getElementByName : ' + getName);
    window.alert('getElementTagName : ' + getTag);
    window.alert('querySelector : ' + query);
    window.alert('querySelectorAll : ' + queryall);

문서 객체의 스타일 조작

style.속성으로 조작 가능

    var getID = document.getElementById('test');
    getID.style.fontSize = '30px';

문서 객체 제거

removeChild(child) : 문서 객체의 자식노드를 제거

    var getID = document.getElementById('test');

    document.body.removeChild(getID);
