기본 내장 객체

자바스크립트에는 많은 기본 내장 객체가 존재, 이중 필요한 것을 잘 찾아 쓰는 것이 중요

1.기본 자료형과 객체의 차이점

  • 자바스크립트 자료형 : 숫자, 문자열, 불리언, 함수, 객체, Undefined
  • 기본 자료형 : 숫자, 문자열, 불리언
        var primitiveNumber = 123;
        var objectNumber = new Number(123);

        alert(typeof(primitiveNumber) + ' : ' + primitiveNumber);
        alert(typeof(objectNumber) + ' : ' + objectNumber);
기본 자료형은 속성 또는 메소드를 추가할 수 없고 객체는 추가할 수 있다.

기본 자료형

        var primitiveNumber = 123;

        primitiveNumber.method = function() {
            alert('abc');
        };

        primitiveNumber.method();

객체

        var objectNumber = new Number(123);

        objectNumber.method = function() {
            alert('abc');
        };

        objectNumber.method();
프로토 타입을 이용한 메서드 추가

객체의 프로토타입에 메소드를 추가하면 기본자료형, 객체 모두 메소드를 사용 가능

        var primitiveNumber = 123;
        var objectNumber = new Number(123);

        Number.prototype.method = function() {
            alert('abc');
        };

        primitiveNumber.method();   // abc
        objectNumber.method();      // abc

Object 객체

Object 객체는 자바스크립트의 최상위 객체

객체 선언

        var object = {};

        var object = new Object();

속성과 메서드

objcet 객체의 메서드
  • constructor() : 객체의 생성자 함수를 나타냄
  • hasOwnProperty(name) : 객체 name 속성이 있는지 확인
  • isPrototypeof (object) : 객체 object의 프로토 타입인지 검사
  • propertyisEnumberable(name) : 반복문으로 열거할 수 있는지 확인
  • toLocaleString() : 호스트 환경에 맞는 언어의 문자열로 변경
  • toString() : 객체를 문자열로 변경
  • valueOf() : 객체의 값을 나타냄

모든 객체에 메서드 추가

Object 객체는 모든 자바스크립트의 최상위 객체이므로 Object 객체의 프로토 타입에 속성 또는 메서드를 추가하면 모든 객체에서 활용 가능하다.

			Object.prototype.test = function() {
				alert(this);

Number 객체

숫자를 표현할 때 사용

Number 객체의 메서드

Object 객체가 가지는 일곱가지 메서드 포함 하여 3가지 메서드를 추가로 갖는다.

  • toExponential() : 숫자를 지수 표시로 나타낸 문자열을 리턴
  • toFixed() : 숫자를 고정 소수점 표시로 나타낸 문자열을 리턴
  • toPrecision() : 숫자 길이에 따라 지수 표시 또는 고정 소수점 표시로 나타낸 문자열을 리턴
			var number = 273.15212456;

			var output='';
			output += number.toFixed(1) + '\n';
			output += number.toFixed(4);
			alert(output);

생성자 함수의 속성

함수는 속성과 메서드를 가질 수 있다.

			function constructor() {}
			Constructor.property = 273;
			Constructor.method - function () {}

			alert(Constructor.property);
Number 객체 메서드
  • MAX_VALUE : 자바스크립트의 숫자가 나타탤 수 있는 최대 숫자
  • MIN_VALUE : 자바스크립트의 숫자가 나타탤 수 있는 최소 숫자
  • NaN : 자바스크립트의 숫자가 나타탤 수 없는 숫자
  • POSITIVE_INFINITY : 양의 무한대 숫자
  • NEGATIVE_INFINITY : 음의 무한대 숫자

String 객체

문자열을 다루는데 사용하는 객체

			var string01 = 'Hello';
			var string02 = new String('Hello');

			var output='';
			output += typeof (string01) + '\n';
			output += typeof (string02);
			alert(output);
String 객체 속성
  • length : 문자열의 길이를 나타냅니다.
String 객체 매서드
  • charAt(position) : 지정한 인덱스 값에 해당하는 문자을 알려줌 (값은 0부터 시작하고 공백도 포함됨)
  • charCodeAt(position) : 지정한 인덱스에 해당하는 문자의 유니코드 번호값을 알려줌
  • indexOf(position) : 앞에서 부터 검색 주어진 문자열의 인덱스 값을 알려줌
  • lastIndexOf(position) : 뒤부터검색 주어진 문자의 인덱스 값을 알려줌
  • substr(start,count) : start 부터 count만큼 문자열을 잘라서 리턴
HTML 관련 메서드
  • big( ) : big 태그로 감싸 리턴
  • small( ) : small 태그로 감싸 리턴
  • blink( ) : blink 태그로 감싸 리턴
  • bold( ) : b 태그로 감싸 리턴
  • fixed( ) : tt 태그로 감싸 리턴
  • italics( ) : i 태그로 감싸 리턴
  • strike( ) : strike 태그로 감싸 리턴
  • sub( ) : sub 태그로 감싸 리턴
  • sup( ) : sup 태그로 감싸 리턴
  • fontsize(크기) : font 태그로 감싸 리턴 크기 조정가능
  • fontcolor(색상) : font 태그로 감싸 리턴 color 지정 가능
  • anchor() : a 태그로 감싸 리턴
  • link("link") : a 태그로 감싸 리턴 href 속성 지정

Array 객체

여러 가지 자료를 쉽게 관리할 수 있게 도와주는 객체 (배열)

생성
  • Array() : 빈 배열을 만든다
  • Array(number) : 매개변추만큼의 크기를 가지는 배열을 만든다
  • Array(any,.. ,any) 매개변추를 배열로 만든다.
Array 객체 속성
  • length : 요소의 개수를 알아냅니다.
Array 객체 메서드
  • cancat() : 매개변수로 입력한 배열의 요소를 모두 합쳐 배열을 만들어 리턴
  • join() : 배열 안의 모든요소를 문자열로 만들어 리턴
  • slice() 요소의 지정한 부분을 리턴
  • --자기 자신을 변화---
  • pop() : 배열의 마지막 요소를 제거하고 리턴
  • push() : 배열의 마지막 부분에 새로운 요소를 추가
  • reverse() : 배열의 요소 순서를 뒤집음
  • sort() : 배열의 요소를 정렬
  • splice() : 요소의 지정한 부분을 삭제하고 삭제한 요소를 리턴

Date 객체

날짜와 시간을 표시하는 객체

			var date = new Date();

			alert(date);
Date객체 날짜와 시간을 읽어오는 메소드
  • getYear( ) : Date객체에서 연도 구함
  • getMonth( ) : Date객체에서 월 구함
  • getDate( ) : Date객체에서 일 구함
  • getDay( ) : Date객체에서 요일 구함
  • getTime( ) : 1970년 이후 경과된 시간을 1/1,000초 단위로 구함
  • getHours( ) : Date객체에서 시 구함
  • getMinutes( ) : Date객체에서 분 구함
  • getSeconds( ) : Date객체에서 초 구함
Date객체 날짜와 시간을 지정하는 메소드
  • setYear( ) : Date객체에서 연도 설정
  • setMonth( ) : Date객체에서 월 설정
  • setDate( ) : Date객체에서 일 설정
  • setDay( ) : Date객체에서 요일 설정
  • setTime( ) : 1970년 이후 경과된 시간을 1/1,000초 단위로 설정
  • setHours( ) : Date객체에서 시 설정
  • setMinutes( ) : Date객체에서 분 설정
  • setSeconds( ) : Date객체에서 초 설정
  • toGMTString( ) : Date객체에서 GMT 기준 날짜 형식으로 알려줌
  • toLocaleString( ) : Date객체에서 날짜와 시간을 "년 월 일 요일 오전/오후 시:분:초" 형태로 알려줌
  • toString( ) Date : 객체의 날짜와 시간을 "요일 월 일 시 시:분:초 연도" 형태로 알려줌

Math 객체

기본 내장 객체중 유일하게 생성자 함수를 사용하지 않는 객체로 수학과 관련된 속성과 메서드를 갖는 객체

객체 생성

Math.메소드()

Math 객체의 속성
  • E : 오일러상수 값
  • PI : 원주율
  • LN10 : 밑수가 10인 상용로그값
  • LN2 : 밑수가 2인 자연로그값
  • SQRT1_2 : 0.5의 제곱근값
  • SQRT2 : 2의 제곱근값
Math 객체의 메서드
  • sin(x) : x의 sine 값 구함
  • cos(x) : x의 cosine 값 구함
  • tan(x) : x의 tangent 값 구함
  • abs(x) : x의 절대값을 구함
  • exp(x) : 지수함수를 구함
  • log(x) : 로그함수를 구함
  • pow(x,y) : x의 y승을 구함
  • sqrt(x) : 제곱근을 구함
  • random( ) : 0에서1사이의 난수를 발생시킴
  • round(x) : x를 반올림한 값 구함
  • floor(x) : x보다 작거나 같은수중 가장큰 정수값 구함
  • ceil(x) : x보다 크거나 같은수중 가장작은 정수값 구함
  • max(x,y) : x와 y중 큰값 구함
  • min(x,y) : x와 y중 작은값 구함
