1. 식별자: 자바스크립트에서 이름을 붙일 떄 사용하는 단어. ex)변수명, 함수명 규칙> 1) 키워드를 사용하면 안됨. 2) 숫자로 시작하면 안됨. 3) 특수문자는 _와 $만 허용. 4) 공백 문자를 포함할 수 없음. 관례> 1) 생성자 함수의 이름은 항상 대문자로 시작합니다. 2) 변수와 인스턴스, 함수, 메서드의 이름은 항상 소문자로 시작합니다. 3) 여러 단위로 이루어진 식별자는 각 단어의 첫 글자를 대문자로 합니다. 자바스크립트 식별자 종류> 1) 식별자 뒤에 괄호 없음: 단독사용: 변수 / 다른 식별자와 사용: 속성 2) 식별자 뒤에 괄호 있음: 단독사용: 함수 / 다른 식별자와 사용: 메서드 문자> \n -> 개행 \t -> 수평 탭 \' -> 작은따옴표 \" -> 큰따옴표 \\ -> 역슬래시 % -> 나머지 ! -> 논리부정연산자 && -> 논리곱 연산자 2. 변수: 값을 저장할 때 사용하는 식별자. 변수 사용법> 1) 변수를 선언 2) 변수에 값을 할당 3. 복합대입 연산자 += : 기존변수값에 더함 -= : 기존변수값에 뻄 *= : 기존변수값에 곱함 /= : 기존변수값에 나눔 %= : 기존변수값에 나머지를 구함 4. if 조건문 if (불표현식){ 문장 } 불표현식이 true이면 문장을 실행하고 false이면 문장을 무시한다. 4-2. if else 조건문 if (불표현식){ 문장a } else{ 문장b } 4-3. if else if 조건문 if (불표현식){ 문장 } else if (불표현식){ 문장 } else if (불표현식){ 문장 } else{ 문장 } 5. switch 조건문 switch (비교할 값){ case 값: 문장 break; case 값: 문장 break; default: 문장 break; }
