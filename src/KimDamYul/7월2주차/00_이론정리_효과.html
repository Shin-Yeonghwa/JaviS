jQuery 효과

jQuery 라이브러리를 사용하면 손쉽게 문서 객체에 효과를 줄 수 있다.
(CSS3로 애니메이션 효과를 적용하면 성능은 좋지만 구 버전의 인터넷 익스플로러에서는 작동하지 않음)

효과 메소드

show(속도) - 문서 객체를 크게 하면서 표시
hide(속도) - 문서 객체를 작게 하면서 표시
toggle(속도) - show() 메소드와 hide() 메소드를 번갈아 실행
slideDown(속도) - 문서 객체를 슬라이드 효과와 함께 보여 표시
slideUp(속도) - 문서 객체를 슬라이드 효과와 함께 사라지게 함
slideToggle(속도) - slideDown() 메소드와 slideUp() 메소드를 번갈아 실행
fadeln(속도) - 문서 객체를 선명하게 하면서 표시
fadeOut(속도) - 문서 객체를 흐리게 하면서 사라지게 함
fadeToggle(속도) - fadeln() 메소드와 fadeOut() 메소드를 번갈아 실행

* 속도를 입력하지 않으면 기본값인 500 밀리초가 적용.
(밀리초 숫자나 slow(600밀리초), normal(500밀리초), fast(400밀리초))

innerfade 플러그인(01_example_innerfade.html)

innerfade 플러그인(화면전환효과): http://medienfreunde.com/lab/innerfade 다운로드
innerfade 플러그인은 ul태그에 적용하는 플러그인, 내부의 li를 자동으로 전환한다.

innerfade() 매서드 옵션

animationtype = 내용물의 변경 효과 - 'fade','slide'
speed = 내용물의 변경 속도 - 'slow', 'normal', 'fast', 숫자
timeout = 변경 효과가 적용되는 속도 - 숫자
type = 내용물의 변경 방식 - 'sequence', 'random', 'random_start'
containerheight = 내용물의 높이 - 'auto', 숫자

사용자 정의 효과 - animate()(02_example_animate.html)

animate() 메서드는 첫 번째 매개변수에 객체가 들어간다는 것을 제외하면
저수준의 시각 효과 메서드와 같은 매개변수 이다.

메서드의 원형 : $(선택자).animate(스타일 [,지속시간] [,시간당속도함수] [,콜백 함수]); 첫번째 매개변수인 객체에 입력할 수 있는 속성은 아래와같다.
- opacity
- height
- top
- width
- left
- margin
- right
- padding
- bottom

+= 연산자나 -= 연산자를 사용하여 상대적 속성 변경 가능

animate() - 애니메이션 큐

jquery의 효과 메서드는 계속 누적된다.
누적된 효과 명령은 큐(queue)에 누적된다.
큐에 있는 내용을 제거하고 싶으면 clearQueue() 메서드를 사용한다.
ex) 
        $(document).ready(function () {

            $('div').animate({ left: '+=60' }, 2000);   

            $('div').animate({ width: '+=60' }, 2000);  
 
            $('div').animate({ height: '+=60' }, 2000);  
 
            //3초후에 함수를 실행

            setTimeout(function (){

               //애니메이션 큐 제거

                $('div').clearQueue();

                //효과메서드 사용

                $('div').hide();

            })

        });

animate() - 애니메이션 정지(02_example_animate_3.html)

애니메이션을 정지하고 싶으면 stop() 메서드를 사용한다.

stop() 매서드의 사용형태

1. $(selector).stop(); - 입력하지 않을 시 자동으로 false 입력한것으로 간주
2. $(selector).stop(clearQueue);
3. $(selector).stop(clearQueue,goToEnd);
