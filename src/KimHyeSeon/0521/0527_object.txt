1. 객체
	var 변수명 = {
		key : 값,
		key : 값,
		key : 값,
		key : 값
	}
- 배열은 인덱스를 사용하지만, 객체는 키를 사용
- 접근 방법은 변수명[key] 또는 변수명.key
- 객체의 키는 식별자 또는 문자열 모두 사용

2. 속성과 메서드
- 객체 내부에 있는 값을 속성이라고 부름
- 객체의 속성 중 함수 자료형인 속성을 메서드라고 부름
- 메서드 내에서 자기 자신이 가진 속성이라는 것을 표시할 때 this를 사용

3. 객체와 반복문
- 객체의 속성을 모두 살펴보려면 for in 반복문을 사용
- for in 반복문에 객체를 넣으면 요소 개수만큼 반복문을 실행

4. 객체 관련 키워드
- in : for 키워드와 별도로 사용하면 해당 키가 객체 안에 있는지 확인할 수 있음
- with : 코드를 짧게 줄여주는 키워드
	width(객체) {}
- with 키워드를 사용하는 객체의 속성 이름과 외부 변수의 이름이 같으면 충돌 발생 (객체 속성을 우선함)

5. 객체의 속성 추가 및 제거
- 동적인 추가 : 객체 변수명.key = 값;
- delete 키워드를 사용 : delete(객체 변수명.key);
- toString() 메서드 : 객체에 있는 속성을 출력하는 메서드
