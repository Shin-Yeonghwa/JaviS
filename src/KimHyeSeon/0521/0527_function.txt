1. 익명 함수
	var 변수 = function() {}
- 함수는 코드의 집합
- 익명함수는 함수 이름이 없으므로 변수에 넣어 사용해야 함

2. 선언적 함수
	function 함수명() {}
- 웹 브라우저는 script 태그 내부의 내용을 한 줄 씩 읽기 전에 선언적 함수부터 읽음
- 익명 함수와 선언적 함수가 함께 쓰인 경우 쓰인 순서와 상관없이 익명 함수를 마지막에 호출함

3. 매개변수와 리턴값
	function 함수명(매개변수, 매개변수) {
		//코드
		return 리턴값;
	}
- 함수 호출시 괄호 안에 적는 것을 매개변수
- 함수를 호출하고 함수가 변환되는 값을 리턴값
- 필요한 경우만 사용

4. 매개변수
- 함수에 선언된 매개변수보다 매개변수를 많이 사용하면 추가된 매개변수는 무시하며,
  적게 사용할 경우 undefined로 입력됨

5. 가변인자함수
- 매개변수의 개수가 변할 수 있는 함수
- 매개변수를 선언된 형태와 다르게 사용했을 때, 매개변수를 모두 활용하는 함수 (예: Array())
- arguments는 매개변수의 배열

6. 리턴값
- 함수가 실행되는 도중에 함수를 호출한 곳으로 돌아가라는 의미
- 값을 지정하지 않아도 함수를 호출한 곳으로 돌아감

7. 내부함수
- 내부함수와 같은 이름의 함수가 있어도 내부 함수가 우선 적용
- 내부함수는 내부함수가 포함되는 함수에서만 사용

8. 콜백함수
- 매개변수로 전달하는 함수
- 익명함수로 사용되는 경우가 많음

9. 함수를 리턴하는 함수
- 클로저를 사용하기 위해 함수를 리턴하는 함수를 사용

10. 클로저
- 클로저를 사용하면 지역변수를 함수 외부에서 사용할 수 있음 (지역변수를 남겨두는 현상)
- 함수로 생성된 공간 또는 리턴된 함수 자체 또는 살아남은 지역변수를 클로져라고 부르기도 함
- 리턴된 클로저 함수를 사용해야 지역변수를 사용할 수 있음
- 클로저 함수로 인해 남은 지역변수는 각각의 클로저 함수의 고유한 변수

11. 내장함수
- 기본적으로 내장됨 함수 (예:alert(), prompt())

11-1. 타이머 함수
- 특정 시간에 : 특정한 함수를 실행
- setTimeout() : 특정 시간 후 함수를 한번만 실행
- setInterval() : 특정 시간마다 함수를 반복해서 실행
- clearTimeout() : 일정시간 후 함수를 한번 실행하는 것을 중지
- clearInterval() : 일정시간마다 함수를 반복하는 것을 중단

11-2. 인코딩과 디코딩 함수
- escape() : 영문 알파벳과 숫자, 일부 특수문자를 제외하고 모두 인코딩
- encodeURI() : escape()함수에서 인터넷 주소에 사용되는 일부 특수문자는 변환하지 않음
- encodeURIComponent() : 알파벳과 숫자를 제외한 모든 문자를 인코딩, urf-8과 동일, 가장많이 사용

11-3. 코드실행함수
- eval(string) : 문자열을 자바스크립트 코드로 실행, 함수로 실행된 코드에서 정의한 변수도 사용가능

11-4. 숫자 확인 함수
isFinite() : number가 무한한 값인지 확인, 유한한 수이면 true를 리턴
inNAN() : number가 NAN인지 확인, NaN은 스스로를 비교할 수 없음

11-5. 숫자 변환 함수
paseInt(string) : 정수로 바꾸어줌
parseFloat(strong) : 유리수로 바꾸어줌
