이벤트

이벤트란 키보드를 이용해 버튼을 입력하거나
마우스 클릭과 같이 다른것에 영향을 미치는것을 의미한다


이벤트 용어 정리

<script type="text/javascript">
        window.onload = function() {};
</script>
  • 이벤트 연결 : window 객체의 onload 속성에 함수 자료형을 할당하는 것
  • 이벤트 이름(타입) : load
  • 이벤트 속성 : onload
  • 이벤트 리스너(핸들러) : 이벤트 속성에 할당한 함수
  • 이벤트 모델 : 문서 객체에 이벤트를 연결하는 방법

이벤트의 종류

    마우스 이벤트
    click (onclick) : 마우스 클릭
    mouseover (onmouseover) : 마우스 포인터가 요소 위에 올라감
    mouseout (onmouseout) : 마우스 포인터가 요소 밖으로 벗어남

    키보드 이벤트
    keydown (onkeydown) : 키보드 버튼 누름
    keyup (onkeyup) : 키보드 버튼 뗌

    HTML 프레임 이벤트
    load (onload) : HTML 문서 및 추가 자원(이미지 등)들을 모두 불러왔을 때 발생
    resize (onresize) : 요소 크기의 변동시
    scroll (onscroll) : 페이지 스크롤 이동

    HTML 입력양식 이벤트
    submit (onsubmit)
    reset (onrest)
    focus (onfocus)
    blur (onblur) : 다른 영역 선택
