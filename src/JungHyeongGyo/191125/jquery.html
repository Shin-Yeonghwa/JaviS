요청방식

GET
자원 조회
POST
자원 추가
PUT
자원 수정
DELETE
자원 삭제
HEAD
자원의 메타 데이터 조회
OPTIONS
사용 가능한 요청 방식 조회
TRACE
테스트 목적의 데이터 조회
CONNECT
연결 요청

상품정보 URL 관리 예시(RESTful 웹)

  • GET 127.0.0.1:52273/products 제품 전체 조회
  • GET 127.0.0.1:52273/products/273 제품 273 조회
  • POST 127.0.0.1:52273/products 제품 추가
  • PUT 127.0.0.1:52273/products/273 제품 273 수정
  • DELETE 127.0.0.1:52273/products/273 제품 273 제거

요청 방식 처리 메서드

app.get()
클라이언트의 GET 요청을 처리합니다.
app.post()
클라이언트의 POST 요청을 처리합니다.
app.put()
클라이언트의 PUT 요청을 처리합니다.
app.del()
클라이언트의 DELETE 요청을 처리합니다.
