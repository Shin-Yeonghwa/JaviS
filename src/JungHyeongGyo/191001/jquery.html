jquery

효과

기본 시각 효과

show()
문서 객체를 크게 하며 보여줍니다.
hide()
문서 객체를 작게 하며 사라지게 합니다.
togle()
show() 메서드와 hide() 메서드를 번갈아 실행합니다.
slideDown()
문서 객체를 슬라이드 효과와 함께 보여줍니다.
slideUp()
문서 객체를 슬라이드 효과와 함께 사라지게 합니다.
slideToggle()
slideDown() 메서드와 slideUp() 메서드를 번갈아 실행합니다.
fadeIn()
문서 객체를 선명하게 보여줍니다.
fadeOut()
문서 객체를 흐리게 사라지게 합니다.
fadeToggle()
fadeIn() 메서드와 fadeOut() 메서드를 번갈아 실행합니다.

사용 예시 : $(selector).method(speed, easing, callback)


speed
효과를 진행할 속도를 지정합니다.
밀리 초 단위의 숫자 또는 문자열 slow, normal, fast를 입력합니다.
callback
효과를 모두 완료하고 실행할 함수를 지정합니다.
easing
애니메이션 easing 형태를 지정합니다.
별도의 플러그인을 사용하지 않으면 문자열 linear와 swing만 입력 가능합니다.

사용자 정의 효과

animate()
사용자 지정 효과를 생성합니다.

$(selector).animate(object, speed, easing, callback);

상대적 애니메이션

현재 상태에서 상대적으로 애니메이션을 적용

예시1 : width: parseInt(width) + 50

예시2 : width: '+=50'

애니메이션 큐

clearQueue()
큐의 내용을 제거합니다.

제이쿼리의 효과 메서드는 계속 누적된다.

애니메이션 정지

stop()
효과 및 애니메이션을 정지

$(selector).stop(clearQueue, goToEnd);

애니메이션 지연

delay()
큐에 있는 명령을 잠시 중지합니다.
